# food_app
```
A new Flutter-Firebase application.
```

## Description
- Place your own ```google-services.json``` file into your app's ```/android/app``` folder.

## Learning
- Google Map
- Google Pay
- Google Auth
- Get Current Location
- State Management using Provider
